# Data Visualization

## If you're "not sure" which library
Use this mental model:
- **Quick dashboards, simple charts** → Recharts / ECharts
- **Highly custom visuals** → D3 / Visx
- **Grammar-of-graphics style** → Observable Plot
- **Large datasets / performance** → ECharts (canvas), WebGL-oriented libs (not listed here)

## Libraries (official)
- D3: https://d3js.org/ and https://github.com/d3/d3/blob/main/API.md
- Observable Plot: https://observablehq.com/plot/
- Recharts: https://recharts.org/en-US/
- Visx: https://airbnb.io/visx/
- Apache ECharts: https://echarts.apache.org/en/index.html
